ภาพรวม
📅
Period
เลือกดูรายวัน / รายเดือน / รายปี ผ่านปุ่มด้านบน
🗓
เดือน/วัน
เลือกเดือนและวันที่ต้องการดูข้อมูล
⌨
Shortcut
กด ← → เพื่อเลื่อนวัน ใน mode รายวัน
แต่ละ Tab
Overall
ภาพรวมทั้งหมด — KPI หลัก 4 ตัว + กราฟรวม 8 อัน เปรียบเทียบทุก operation
Inbound
การรับสินค้า — Plan vs Actual รายวัน แหล่งที่มา (DC) Productivity HC และยอดสะสม
Out SO
จ่ายสินค้าลูกค้า (Sales Order) — MT / TT / Export สัดส่วน จำนวนรถ Productivity
Out STO
โอนย้ายระหว่าง DC (Stock Transfer Order) — Dispatch แยก DC Truck HC ยอดสะสม
Stock
สต็อกคลัง — Inventory vs Capacity Utilization % gauge Bin Location
Table
ตารางข้อมูลดิบ — กรองตาม operation ดูรายละเอียดทุก field Export CSV ได้
ความหมายสี
สีเขียว = Hit Plan
Actual ≥ Plan / ปกติ / ยอดสะสมนำแผน
สีแดง = Miss Plan
Actual < Plan / เกิน Capacity / ต่ำกว่า Target
สีส้ม = เฝ้าระวัง
Utilization 80–90% / ใกล้แผน ±10%
สีน้ำเงิน = ข้อมูลหลัก
กราฟหลัก / บาร์ Actual ทั่วไป
กราฟพิเศษ
📈
Cumulative + Gap Fill
เส้นยอดสะสม Actual vs Plan — พื้นที่สีเขียวคือ "นำแผน" สีแดงคือ "ตามหลัง" เส้นประคือ Projected ถึงสิ้นเดือน
🎯
Productivity + Red Zone
เส้น C/MH vs Target — พื้นที่สีแดงโปร่งใสระหว่างเส้น actual กับ target คือช่วงที่ต่ำกว่าเป้า
🌡
Stock Danger Zone
กราฟ Inventory — แถบสีส้ม 80–90% และแดง 90–100% ของ Capacity แสดงช่วงวิกฤต
🔵
Gauge Cases/Bin
การ์ด Utilization % — แถบ progress พร้อม tick marks ที่ 80% และ 90% บอก threshold ทันที
Alert & ปุ่มต่างๆ
🔴 Alert Bar
แถบแดงด้านบน แสดงเมื่อมี operation ผิดปกติ เช่น IB/SO/STO ต่ำกว่าแผน >10% หรือ Stock Util >90%
📸 Capture
ถ่ายภาพหน้าจอ dashboard เพื่อแนบรายงาน
🌙 Dark/Light
สลับธีมสว่าง/มืด ระบบจำค่าไว้ครั้งต่อไป
📥 Export CSV
ใน Tab Table กดเพื่อ download ข้อมูลเป็น .csv
